## UI Component Tree

```
Card
├── Badge
├── SectionHeader
├── Avatar + StatusIndicator
└── Content Area

EmployeeCard (extends Card)
├── Avatar
├── Employee Name & Code
├── Badge (Role)
└── Location & Date

ContactCard (extends Card)
├── Avatar
├── Business Name
├── Badge (Category)
├── Location & Phone
└── GST Display
```

## Data Flow

```
Screen Component
    │
    ├─ useAppColors() hook → Colors
    ├─ useState() → Local state
    └─ Mock Data / API Service
         │
         ├── EmployeeCard
         │    └── Avatar, Badge, StatusIndicator
         │
         ├── ContactCard
         │    └── Avatar, Badge
         │
         └── Node (Hierarchy)
              └── Avatar, Badge
```

## Design System

```
Colors (Light/Dark)
├── Primary: Indigo 600
├── Background: Slate 50 / Slate 950
├── Surface: White / Slate 900
├── Text: Slate 900 / White
└── Status: Success, Warning, Danger, Info

Spacing (8-point)
├── xs: 4px
├── sm: 8px
├── md: 16px
├── lg: 24px
├── xl: 32px
└── 2xl: 48px

├── Display: 32
├── H1: 28

├── H3: 20

```
